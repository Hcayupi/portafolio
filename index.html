DinoData - Diccionario de dinosaurios

Descripción:

Sitio Web permite la búsqueda de un dinosaurio y conocer sus principales caracteristicas.

Desafío:

El principal desafío fue la obtención de datos completos y consistentes (nombre, descripción e imagen) de cada dinosaurio, ya que las APIs públicas disponibles resultaban limitadas e insuficientes para los requerimientos del proyecto.

Solución:

Se identificó una fuente externa con información detallada de cada dinosaurio y se investigó la forma de estructurar dichos datos para su uso en la aplicación. A partir de esto, se construyó una API REST utilizando ExpressJS, basada en archivos JSON, permitiendo centralizar y exponer la información mediante endpoints para su consumo desde el frontend.

Tecnologías:

ExpressJS, VueJS y TypeScript

Aprendizajes:

Implementación de una arquitectura cliente-servidor mediante una API REST con ExpressJS, manejo de datos estructurados en JSON, definición de endpoints y rutas dinámicas, y consumo de servicios desde el frontend con VueJS. Uso de TypeScript para tipado estático, mejorando la calidad y mantenibilidad del código.
